
Columns are:

1. count* (count within a trial -same parameters)
2. Time* of key pressing or lease (time of an event)
3. Key number* (1 if key "->" pressed, 2 if realeased; -1 if key "<-" pressed and -2 if released)
4. luminance interesections.
5. luminance in other units.
6. Vel grating 1
7. Vel grating 2
8. Orientation 1 (10)
9. Orientation 2 (180-10 = 170)
10. duty_cycle_1 (see bellow for lambdas, it is the same for duty cycles)
11. duty_cycle_2, 
12. duty_cycle_A 
13. duty_cycle_B
14. i_back (it is the global orientation of each grating: grating with the unfixed
	   parameters- 
           moving up rigth (0)
           moving up left (1), 
	   moving down right (2), 
           moving down left (3))					
15. lambda_A*, (A corresponds to the unfixed parameter grating)
16. lambda_B*, (B correspond to the fixed parameter grating, here wavelegnth=0.15)
17. lambda_1*  (wavelength of the grating moving to the right)
18. lambda_2*, (wavelength of the grating moving to the left)
19. orientation_1_temp (some global orientation)
20. orientation_2_temp
